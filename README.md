# Calculator
## This application is a basic arithmetic calculator
-Created using HTML, CSS, JavaScript
## Functions
-Add, Subtract, Multiply, Divide
-Clear screen
-Delete previous element
-Convert displayed number into percentage
## Goals
The goal of this project was to get familiar with JavaScript and UI/UX. My primary focus was creating a front-end that was visually appealing and functional at the same time. Every button has some sort of feedback and the color scheme is very easy on the eyes
## Fixes
The application currently takes input of multiple consecutive non-number characters such as decimals. Nothing will display on the screen if the user presses enter, however I will be working on preventing the user from inputting unecessary characters for the purpose of user experience.
